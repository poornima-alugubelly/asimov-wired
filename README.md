# Asimov-Wired

Asimov Wired is a sc-fi themed social media app built using React,Redux,chakra UI and Mockbee (mock backend)

## Features Present

- Authentication
  - Login
  - Logout
  - Signup
- Create a Post
  - Like
  - Comment & Replies
- Edit & Delete Post
- User Feed
  - Sort by Date and Trending
- User Profile
  - Add Profile Picture
  - Bio
  - Portfolio URL
- Follow/Unfollow
- Search by
  - User firstname or lastname
  - username
- Explore Feed
- Bookmark Post
- Loading & Alerts
- User Profile Page
